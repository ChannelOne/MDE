# MDE
MDE(Markdown Editor) is a Markdown editor based on Electron, written in Typescript.

- ✔ String datastructure(Lines Chains)
- ✔ Basic UI
- ✔ Text presentation and selectable
- ✔ Text Editable
- ✔ Syntax Highlighting
- ✔ Edit history
- ✔ Render Markdown
- ✔ White/Black Color Theme
